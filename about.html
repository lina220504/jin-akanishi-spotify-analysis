📊 データ・分析について

🎵 データソース

  • API: Spotify Web API
  • アーティスト: 赤西仁 (Jin Akanishi)
  • データ収集日: 2025年06月30日
  • 分析楽曲数: 128曲

📈 分析指標

  • 人気度スコア: Spotify独自の0-100スケール（リアルタイム更新）
  • リリース日: Spotify上での配信開始日
  • 楽曲長: ミリ秒単位の楽曲時間
  • マーケット数: 楽曲が利用可能な国・地域数

⚠️ 制限事項

  • 音響特徴（Danceability, Energy等）は API制限により取得不可
  • 人気度スコアは相対的指標（絶対的な再生数ではない）
  • 地域制限により一部楽曲が対象外の可能性
  • Spotify上で配信されていない楽曲は含まれない

🔧 技術スタック

  • データ分析: Python (pandas, numpy)
  • 可視化: Plotly
  • API: Spotipy (Spotify API wrapper)
  • 実行環境: Google Colaboratory
  • ホスティング: GitHub Pages

📊 キャリア期間分類

  • ソロ初期 (2011-2012): KAT-TUN脱退後の初期ソロ活動
  • 確立期 (2013-2015): ソロアーティストとしての確立
  • 独立期 (2016-2018): 独立後の音楽活動
  • 成熟期 (2019-2021): アーティストとしての成熟
  • 近年 (2022-2025): 最新の音楽活動
